What makes a Goober?

A Goober is a hand-drawn cartoon dog with a simple seated shape. The base design stays recognizable, but the personality changes with every version.

  • Simple rounded body shape
  • Long snoot and big simple nose
  • Hand-drawn, imperfect lines
  • Named by style, costume, mood, or chaos level

Add a Goober

Use this lower-page upload tool to add new Goobers after you review the gallery. Pick an image, give it a name, choose a category, and it will appear above after upload.

This version is set up for Cloudflare. New Goobers are sent to a Worker API, the image is stored in R2, and the Goober details are saved in D1.
